- **Content:** 4 (16px) within sections

### Shadows
- **Cards:** `shadow-sm` (subtle)
- **Modals:** `shadow-xl` (prominent)
- **Hover:** Slightly increased shadow

## ðŸš€ User Workflows

### Workflow 1: Add Company Logo
```
1. Click "Upload Company Logo" or "Replace Logo"
2. Modal appears with drag-drop area
3. Drag logo file or click "Choose File"
4. See preview of logo
5. Click "Upload"
6. Logo appears in identity tab
```

### Workflow 2: Set Business Hours
```
1. Go to "Business Hours" tab
2. For each day:
   - Enter opening time
   - Enter closing time
   - OR toggle "Closed"
3. Click Save
```

### Workflow 3: Add Multiple Languages
```
1. Go to "Languages" tab
2. Click "+ Add Language"
3. Select language from dropdown
4. Enter/translate company info
5. Switch between tabs to enter other languages
6. (Optional) Copy from English, then translate
7. Click Save
```

### Workflow 4: Add Social Media Links
```
1. Go to "Social Media" tab
2. Click "+ Add Facebook"
3. Enter Facebook URL
4. Repeat for other platforms
5. Click Save
```
